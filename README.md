# Bank Loan Risk Analysis Project

## 📊 Project Overview
This project focuses on analyzing bank loan data to manage lending risk effectively. It combines Python, SQL, and Power BI to extract insights and build interactive dashboards.

---

## ⚙️ Workflow

### 1. Python Data Processing
- Data obtained from Microsoft dataset (text format)
- Converted text → CSV using Python (`txt_to_csv_converter.py`)
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Data quality checks
- Data visualization using Matplotlib
- Final selection of 5 key insights

### 2. SQL Analysis
- Schema creation (`schema.sql`)
- Loan analysis queries (`bank_loan_queries.sql`)
- Structured data exploration using MySQL

### 3. Power BI Dashboard
- Executive Overview dashboard
- Risk Analysis dashboard
- Loan portfolio insights visualization

## 📌 Key Insights

- Employment history strongly impacts default risk
- Higher utilization leads to higher default rates
- Loan grade is a strong risk indicator
- Purpose of loan affects default probability
- Debt consolidation holds major loan volume share
